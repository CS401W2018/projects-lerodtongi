1.Internet

The internet is just a network that able to connect the entire world. It can be accessed by anyone at anytime. It has shaped this world unlike any other invention

2.Web

The web is used to access and to upload documents and media sources of any kind.

3.How are the Internet and Web different?

Basically the web uses the internet to store documents and information that can be accessed around the world. The webs relationship to the internet is that of a phone and a person. The phone is the machine but the person uses it to its fullest potential.

4.ASCII and Unicode

These are both languages on a computer. ASCII uses 7 bits to assign a number to 128 characters. Unicode uses different ways to assign numbers to a much larger amount of characters.

5.What is UTF-8?

It is a way to store symbols and characters. It uses Unicode to do this. It is actually also great with ASCII files as well.

6.URL

These are basically addresses to the web browsers. They tell the internet exactly where a web page is located.

7.Domain Name

Domain names are basically the same as URLs. When we input a URL the computer reads a bunch of numbers. But to us they are letters. Ex: the domain name for google is "google.com'

8.HTTP

It is the language used by browsers and servers to talk to each other and trade information

9.HTTPS

HTTPS is the same thing but the S stands for Secure. It is more encrypted and safer. It is usually used with websites that are given sensitive information such as bank cards, social security, etc.

10.IP Address

An IP Address is an address for computers, servers or even websites. They are only made up of numbers and are mostly used as a "computer language".

11.DNS

Stands for Domain Name System. It is compareable to a phonebook for the internet. It is what the machines use to translate the human language into machine language.

12.W3C Recommendations

W3C is an organization that is in charge of web technology. They make sure the web is running smoothly. The reccomendations of this organization is a set guidlines and standards.

13.XML,XHTML,HTML,HTML5

These are all languages used to create web pages.
