Explain the difference between an absolute link and a relative link. Why would you use relative links?

    The difference between these two types of link are that absolute is for links that lead to outer sources that are not in the domain. relative links are for links within the domain. Relative links are shorter and more flexible for navigating a website.

List the 3 rules with relative links.

  1. Same Folder

    2. Just change the file name.

  2. Different Root

    3. begins with ../

  3. Start at Root

    4. Start over

Do links always need to be inside a NAV tag?

They don't always have to be in a NAV tag. Links outside of NAV tags serve a different purpose other than navigation.
